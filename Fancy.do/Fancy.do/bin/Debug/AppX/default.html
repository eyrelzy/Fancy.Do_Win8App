Fancy.Do Help:

How to switch pages? Click navigation on the bottom right.
Use  ← →↑ ↓ 
Esc to show preview for all the pages.
Tablet PC supports navigation only.
In each view, swipe left or right to next page.

How to add a task to do? Click(tab on tablet PC) the '+' block or an empty block or click template task blocks.

How to make custom design template tasks? We already have two template tasks, you may input other tasks to update your template task library.
Swipe up to check all and swipe left on each item to delete, down to cancel.

How to check all your to-do list?
Swipe up to add a new location in Location View, down to cancel. So does the File View.
The tasks shown are colored referred to level settings.

Click each title to return.
